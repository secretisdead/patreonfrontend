Because Patreon currently doesn't support accessing other users' campaign information through a third party client, you must first create your own client on Patreon, and then connect it by entering the information below.

If you have an existing client already created for other purposes, you should still create a new client to use solely for site Patreon integration.

You only need to create one of these clients per Patreon creator account, because this single client will retrieve all capaign information for that account.

This is currently set up for admins to manage everything, but it should be possible to modify this package to allow individual users in a multi-user instance to manage their own clients without seeing each others' clients if necessary.
